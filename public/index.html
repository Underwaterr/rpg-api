RPG API

Your quest for a public RESTful API is over!

Featuring fun lil graphics, item descriptions and stats that don't mean anything.

No, this RPG does not actually exist. But it could!

How to Use

Send HTTP GET requests to http://rpg-api.com/[resource]/[id] where [resource] is one of the available resources and [id] is an optional parameter for getting a specific item.

For example, curl http://rpg-api.com/weapons will get you all weapons and curl http://rpg-api.com/weapons/19 will get you a bow.

Available Resources

  • /weapons
  • /armor
  • /accessories
